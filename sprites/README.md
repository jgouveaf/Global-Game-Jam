# Sprites
Coloque aqui os sprites (.png) para o jogo.
Formatos aceitos: PNG com fundo transparente.

## Como usar
Basta colocar o arquivo .png nesta pasta.
O jogo vai carregar automaticamente via `new Image()` e usar `ctx.drawImage()`.

## Nomes sugeridos
- `barn.png` — Celeiro
- `house.png` — Casa
- `windmill.png` — Moinho
- `tree.png` — Árvore
- `crops.png` — Plantações
- `fence.png` — Cerca
- `well.png` — Poço
- `river.png` — Rio / Água
- `ground.png` — Tileset de chão
- `hud.png` — Interface
