# Sprites
Coloque aqui os sprites (.png) para o jogo.
Formatos aceitos: PNG com fundo transparente.

## Como usar
Basta colocar o arquivo .png nesta pasta.
O jogo vai carregar automaticamente via `new Image()` e usar `ctx.drawImage()`.
